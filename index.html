Dokumentasi Rest Api Mahasiswa

    Tools Yang Perlu Di Persiapkan
  1. XAMPP
  2. Postman
  3. Project Laravel

1. XAMPP

Pastikan anda sudah mendownload atau menginstal XAMPP di komputer atau laptop anda. Dan jika anda sudah menginstall XAMPP nya silahkan anda start Apache dan MySQL nya seperti gambar dibawah ini.

2. Postman

Lalu yang kedua anda harus menginstall Postman untuk memudahkan kita dalam mengambil data api nya. Anda bisa mendownloadnya di browser. Jika sudah terdownload maka buka postmannya dan pastikan anda sudah register agar bisa di gunakan seperti gambar dibawah ini.

3. Project Laravel

Lalu yang ketiga anda harus sudah mendownload project laravelnya. Anda bisa mendownload untuk versi terbaru

Pertama setting file .env dengan nama database nya seperti dibawah ini

Lalu buka terminal dan ketikkan php artisan make:model Mahasiswa -m perintah ini berfungsi untuk membuat sebuah Model dan Migration pada laravel

Lalu setelah itu buka file migration yang berada di database/migrations dan file model yang berada di app/models dan setting seperti gambar dibawah ini

Lalu setelah itu buatlah satu database dengan nama rest_api_mahasiswa seperti di file .env tadi. Setelah itu ketikkan perintah php artisan migrate di terminalnya.

Seteleh itu buatlah folder dengan nama Helpers dan di dalamnya buat file dengan nama ApiFormatter.php lalu ketik syntax seperti gambar dibawah ini.

Lalu ketik perintah seperti dibawah ini di terminal anda. Nanti bakalan dibuatkan sebuah file controlller dengan resource-nya

Lalu ketikkan perintah berikut di dalam file MahasiswaControlller

Lalu cari file api.php di dalam folder route dan ketikkan syntax seperti dibawah ini.

Oke kita sudah membuat sebuah Rest API nya, maka sekarang mari kita coba di Postman dengan melakukan CRUD.

Hal yang perlu diperhatikan adalah endpoint atau url dan method nya disaat menggunakan postman untuk testing.

Buka postman nya lalu tombol tambah dan isi url atau endpointnya beserta methodnya POST. seperti gambar dibawah ini, dan hasilnya seperti hasil yang paling bawah yang fungsinya menambah data.
